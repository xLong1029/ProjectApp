# projectApp

项目申报资讯平台App测试开发

## 分支介绍

<p>
master：线上主分支，保存打包后的文件<br/>
develop: 线上开发分支，保存开发文件
</p>

<p>
app_master：移动app主分支，保存打包后的文件<br/>
app_develop: 移动app开发分支，保存开发文件
</p>

<p>
v1.0: 1.0版本分支<br/>
v1.1: 1.1版本分支<br/>
v1.2: 1.2版本分支
</p>
